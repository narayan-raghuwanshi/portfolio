## Getting started
``` bash
git clone https://github.com/narayan-raghuwanshi/portfolio.git
```
``` bash
npm install
```
``` bash
npm run dev
```
### Visit [http://locahlost:3000/](http://locahlost:3000/)

It is all set, now try changing anything in the code and it will reflect on the browser.
